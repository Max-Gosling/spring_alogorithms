# Задача на сортировку

В данной задаче необходимо реализовать сортировку одним из эфективных алгоритмов

* Merge sort
* Quick sort
* Heap sort

## Формат ввода

```
N
a1 a2 ... aN
```

- `N` — количество элементов
- `a1 ... aN` — целые числа

## Формат вывода

```
a1 a2 ... aN
```

Элементы, отсортированные по неубыванию.

## Примеры

| Ввод | Вывод |
|------|-------|
| `5`<br>`5 3 1 4 2` | `1 2 3 4 5` |
| `6`<br>`3 3 1 1 2 2` | `1 1 2 2 3 3` |
| `7`<br>`-3 5 0 -1 10 -7 3` | `-7 -3 -1 0 3 5 10` |
