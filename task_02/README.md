# Поиск границы 0 → 1

Дан массив из нулей и единиц. Первый элемент всегда `0`, последний всегда `1`.
Необходимо найти любой индекс `i` такой, что `a[i] = 0` и `a[i+1] = 1`.

## Формат ввода

```
N
a1 a2 ... aN
```

- `N` — размер массива (N ≥ 2)
- `a1 ... aN` — элементы массива (только `0` и `1`, первый = `0`, последний = `1`)

## Формат вывода

```
i
```

`i` — индекс (0-индексация), такой что `a[i] = 0` и `a[i+1] = 1`.
Если подходящих индексов несколько, вывести любой из них.

## Примеры

| Ввод | Вывод |
|------|-------|
| `2`<br>`0 1` | `0` |
| `5`<br>`0 0 0 1 1` | `2` |
| `6`<br>`0 0 0 0 0 1` | `4` |
| `5`<br>`0 1 1 1 1` | `0` |
