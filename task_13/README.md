# Расписание уроков

Помогите Алле написать код алгоритма для выбора уроков, которые пройдут в классе.
Дано расписание предметов. Нужно составить расписание, в соответствии с которым
в классе можно будет провести как можно больше уроков.

## Формат ввода

```
N
s1 e1
s2 e2
...
sN eN
```

- `N` — количество предметов (не превосходит 1000)
- `si ei` — время начала и окончания урока (вещественные числа, указываются только значащие цифры, разделитель дробной части — точка)

## Формат вывода

```
K
s1 e1
s2 e2
...
sK eK
```

Первая строка — количество выбранных уроков. Далее K строк — выбранные уроки в порядке возрастания времени начала.

## Примеры

| Ввод | Вывод |
|------|-------|
| `5`<br>`9 10`<br>`9.3 10.3`<br>`10 11`<br>`10.3 11.3`<br>`11 12` | `3`<br>`9 10`<br>`10 11`<br>`11 12` |
