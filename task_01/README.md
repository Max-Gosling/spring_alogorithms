# Задача 1

Дано целое число и отсортированый массив целых чисел, нужно найти 2 числа из массива которые в сумме дадут заданное число

## Формат ввода

```
S
N
a1 a2 ... aN
```

- `S` — целевая сумма
- `N` — размер массива
- `a1 ... aN` — отсортированный по возрастанию массив целых чисел

## Формат вывода

```
x y
```

Два числа из массива, сумма которых равна `S` (в порядке возрастания).
Если решения нет — вывести `-1`.

## Примеры

| Ввод | Вывод |
|------|-------|
| `9`<br>`4`<br>`2 7 11 15` | `2 7` |
| `6`<br>`3`<br>`1 2 4` | `2 4` |
| `100`<br>`3`<br>`1 2 3` | `-1` |
