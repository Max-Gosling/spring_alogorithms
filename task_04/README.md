# Задача на стэк

В данной задаче необходимо реализовать один стэк и стэк с минимумом (нельзя использовать std::stack). Получение минимума должно работать за константное время.

## Интерфейс

### Stack

```cpp
class Stack {
  void Push(int value);  // положить элемент на вершину
  int Pop();             // снять и вернуть верхний элемент
};
```

### MinStack

```cpp
class MinStack {
  void Push(int value);  // положить элемент на вершину
  int Pop();             // снять и вернуть верхний элемент
  int GetMin();          // вернуть минимальный элемент за O(1)
};
```

## Тестирование

Задача проверяется через unit-тесты (gtest). Тестовые файлы stdin/stdout не используются.
