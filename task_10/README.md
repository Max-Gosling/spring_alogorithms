# Задача на хэш таблицу

В данной задаче необходимо реализовать хэш таблицу.

## Интерфейс

```cpp
void Insert(int key, int value);   // вставка пары ключ-значение
int Get(int key);                  // получение значения по ключу
void Remove(int key);              // удаление по ключу
bool Contains(int key);            // проверка наличия ключа
```

Операции в среднем должны работать за O(1).

## Тестирование

Задача проверяется через unit-тесты (gtest). Тестовые файлы stdin/stdout не используются.
