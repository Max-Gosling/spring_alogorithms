# Задача на количество дней перед потеплением

Написать функцию в которую передается темепература за каждый день в определенный момент день, нужно вернуть количество дней до повышения температуры для каждого дня, если температура не повысится, то для этого дня поставить в результирующий массив 0.

## Формат ввода

```
N
t1 t2 ... tN
```

- `N` — количество дней
- `t1 ... tN` — температура в каждый день

## Формат вывода

```
d1 d2 ... dN
```

`di` — количество дней, через которое температура станет выше, чем в день `i`. Если потепления не будет — `0`.

## Примеры

| Ввод | Вывод |
|------|-------|
| `8`<br>`73 74 75 71 69 72 76 73` | `1 1 4 2 1 1 0 0` |
| `4`<br>`30 40 50 60` | `1 1 1 0` |
| `4`<br>`60 50 40 30` | `0 0 0 0` |
