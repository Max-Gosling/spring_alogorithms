# Задача на дерево поиска

В данной задаче необходимо реализовать дерево поиска со сложностью O(log(n)).

## Интерфейс

Реализовать сбалансированное дерево поиска (AVL / Red-Black / Treap и т.д.) с операциями:

```cpp
void Insert(int key);    // вставка ключа
bool Find(int key);      // поиск ключа
void Remove(int key);    // удаление ключа
```

Все операции должны работать за O(log(n)).

## Тестирование

Задача проверяется через unit-тесты (gtest). Тестовые файлы stdin/stdout не используются.
