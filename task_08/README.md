# Задача на порядковые статистики

В данной задаче необходимо реализовать поиск n-ой порядковой статистики.

## Формат ввода

```
N K
a1 a2 ... aN
```

- `N` — количество элементов в массиве
- `K` — номер порядковой статистики (1-индексация)
- `a1 ... aN` — целые числа

## Формат вывода

```
x
```

`x` — K-й по величине элемент (1 = минимум, N = максимум).

## Примеры

| Ввод | Вывод |
|------|-------|
| `5 3`<br>`7 10 4 3 20` | `7` |
| `5 1`<br>`7 10 4 3 20` | `3` |
| `5 5`<br>`7 10 4 3 20` | `20` |
