# Подпоследовательность

Даны строка символов и подпоследовательность. Надо понять, есть ли в строке непрерывная подпоследовательность, являющаяся перестановкой данной подпоследовательности и если да, вернуть номер символа, с которого начинается подстрока. Если нет, вернуть -1.

## Формат ввода

```
S
P
```

- `S` — строка символов (латинские буквы в нижнем регистре)
- `P` — подпоследовательность для поиска

## Формат вывода

```
i
```

`i` — номер символа (1-индексация), с которого начинается подстрока-перестановка. Если не найдено — `-1`.

## Примеры

| Ввод | Вывод |
|------|-------|
| `abcsrlfcxdms`<br>`lrf` | `4` |
| `smnkfwlvk`<br>`nmsf` | `-1` |
| `abcabc`<br>`cab` | `1` |
