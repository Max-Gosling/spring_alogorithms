# Задача о размене монет

Дано число и вектор с номиналами монет, нужно вернуть минимальное количество монет которое можно набрать данными номиналами.

## Формат ввода

```
S
N
c1 c2 ... cN
```

- `S` — целевая сумма
- `N` — количество номиналов
- `c1 ... cN` — номиналы монет

## Формат вывода

```
K
```

`K` — минимальное количество монет для набора суммы `S`. Если набрать невозможно — `-1`.

## Примеры

| Ввод | Вывод |
|------|-------|
| `14`<br>`4`<br>`1 2 5 10` | `3` |
| `19`<br>`4`<br>`1 2 5 10` | `4` |
| `3`<br>`1`<br>`2` | `-1` |
